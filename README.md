# SparseDP-LCSubstr
Check the github page (https://o-logn.github.io/SparseDP-LCSubstr) for an introduction.\
Check the preprint (https://doi.org/10.5281/zenodo.18817204) for a more rigorous explanation, and an i.i.d. analysis.

# Dependencies
In `sparse_dp.py` and `sparse_dp_k.py` i used python's dataclass module, which requires Python 3.7+. However, a standard class suffices.
To run the benchmark, the suffix tree code of https://github.com/cceh/suffix-tree was used.
